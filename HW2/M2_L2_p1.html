Gradient function

We need to define a function for the gradient of our objective function. In the lecture, this gradient was: $$\frac{\partial \text{obj}}{\partial w} = 2X' (X w - y)$$

Please complete the function below to compute the gradient. Recall that the design matrix is the global variable X, and you may need to reshape the output y as a column vector.

def grad(w):
    # YOUR CODE GOES HERE
    # Return the gradient of the objective function
    w_grad = 2*X.T @ (X@w - y.reshape(-1,1))
    return w_grad
